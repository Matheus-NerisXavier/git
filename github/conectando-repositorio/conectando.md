## Conectando Repositório local com repositório remoto

- Repositórios
- Criar um novo repositório
- Colocar o nome do seu repositório
- Selecionar o tipo do repositório (publico ou privado)
- Adicionar read.me (opcional)
- Adicionar o arquivo .gitignore (opcional)

- Copiar as 3 linhas .. or push an existing repository from the command line
- Colar essas linhas no terminal
